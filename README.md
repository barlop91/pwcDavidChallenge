# pwcDavidChallenge

## Prerequisites

Before running the project, make sure you have:

- Python installed
- Google Chrome installed
- Required dependencies installed from `requirements.txt`

## Setup Instructions

### 1. Clone the repository

git clone https://github.com/barlop91/pwcDavidChallenge.git
cd pwcDavidChallenge

## How to run

UI test:
python -m pytest tests/

API test:
python -m pytest api/
